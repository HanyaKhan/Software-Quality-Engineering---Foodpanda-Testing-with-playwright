# Software_Quality_Engineering (Foodpanda_Testing_with_playwright)

## Foodpanda End to End Testing Project using Playwright

### Overview

This project demonstrates automated end to end testing of the Foodpanda web application using Playwright. The purpose of this project is to validate the core user journey, starting from searching for menu items to placing an order and completing the checkout process.

### Features Covered

The automated test cases in this project cover the following scenarios:

* Searching for restaurants and menu items
* Viewing and selecting food items
* Adding items to the cart
* Modifying order details
* Proceeding to checkout
* Placing an order successfully

### Tools and Technologies

* Playwright for browser automation
* JavaScript or TypeScript for scripting
* Node.js as the runtime environment

### Test Approach

All test cases are written to simulate real user interactions. The tests focus on reliability, proper element handling, and validation of expected outcomes at each step of the ordering process.

### Project Structure

* Test files containing all automated scenarios
* Page objects or reusable functions for better maintainability
* Configuration files for Playwright setup

### How to Run the Project

1. Clone the repository
2. Install dependencies using `npm install`
3. Run tests using `npx playwright test`

### Purpose

This project is intended for learning and demonstrating automation testing skills, particularly in end to end testing of a real world application workflow.

### Repository

The complete source code and test cases are available in this repository.
